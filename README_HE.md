מערכת מקיפה ל**ניתוח נטישת לקוחות** לחברות טלקום עם **בינה מלאכותית מתקדמת**, **צינור ETL אוטומטי**, **ויזואליזציות אינטראקטיביות** ו**דוחות מנהלים מקצועיים**.

## ✨ תכונות מהפכניות

### 📊 **צינור ETL מלא ואוטומטי**
- 🔄 **חילוץ חכם** - CSV, Excel, APIs חיצוניים ונתוני סימולציה
- 🧹 **טרנספורמציה מתקדמת** - ניקוי, נורמליזציה והנדסת פיצ'רים
- ✅ **ולידציה חזקה** - כללי עסקיים וזיהוי חריגות
- 💾 **טעינה רב-פורמטית** - ייצוא ל-CSV, Excel, Parquet ו-JSON

### 🤖 **בינה מלאכותית ולמידת מכונה**
- 🎯 **קיבוץ K-Means** - פילוח לקוחות אוטומטי
- 📈 **חיזוי ARIMA** - חיזוי מגמות נטישה
- 🔍 **Isolation Forest** - זיהוי התנהגויות חריגות
- 📊 **ניתוח סטטיסטי** - Chi-Square, T-Test, ANOVA

### 🌐 **אפליקציית אינטרנט פרוגרסיבית (PWA)**

#### 📱 **Progressive Web App עם תמיכה במצב לא מקוון**
- 💫 **PWA מלאה** - ניתנת להתקנה כאפליקציה מקורית בכל מכשיר
- 🔌 **מצב לא מקוון** - עובד לחלוטין ללא חיבור לאינטרנט
- ⚡ **Service Worker** - קישינג חכם של משאבים קריטיים
- 🎨 **דשבורד אינטראקטיבי** - ויזואליזציות בזמן אמת עם Plotly.js
- 📊 **כרטיסי KPI** - מדדים מרכזיים גלויים מיידית
- 🌍 **רב-לשוני באינטרנט** - החלפת שפה דינמית (7 שפות נתמכות)
- 📱 **עיצוב רספונסיבי** - התאמה מושלמת לנייד, טאבלט ושולחן עבודה
- 🎯 **עמוד לא מקוון אלגנטי** - חוויית משתמש מקצועית ללא חיבור
- 🔄 **חיבור מחדש אוטומטי** - זיהוי אוטומטי של שחזור רשת
- 💾 **קישינג אסטרטגי** - נכסים קריטיים נטענים מראש לביצועים אופטימליים

#### 💻 **כיצד להשתמש בממשק האינטרנט**
```bash
# 1️⃣ נווט לתיקיית web
cd web/

# 2️⃣ הפעל שרת מקומי
python -m http.server 8000

# 3️⃣ פתח בדפדפן
http://localhost:8000

# 4️⃣ התקן כ-PWA (אופציונלי)
# Chrome: לחץ על ⊕ בשורת הכתובת → "התקן Telecom X"
# Edge: לחץ על "..." → "אפליקציות" → "התקן את האתר הזה"
# Safari: לחץ על "שתף" → "הוסף למסך הבית"

# 5️⃣ בדוק מצב לא מקוון
# DevTools (F12) → Application → Service Workers → ☑️ "Offline"
# ✅ האפליקציה תמשיך לעבוד ותציג את offline.html
```

#### 🎯 **מבנה קבצי PWA**
```
web/
├── 📄 index.html          # דשבורד ראשי עם KPIs
├── 📄 offline.html        # עמוד לא מקוון אלגנטי (9 KB)
├── 📄 manifest.json       # מניפסט PWA
├── 📄 sw.js              # Service Worker עם קישינג
├── 📂 css/               # עיצובים (main, components, animations)
└── 📂 js/                # סקריפטים (main, language-manager, charts)
```

### 🌍 **מערכת רב-לשונית מלאה**

#### 🗣️ **שפות נתמכות**
- 🇪🇸 **ספרדית** - שפה ראשית עם לוקליזציה מלאה
- 🇺🇸 **אנגלית** - תמיכה בינלאומית לשוק גלובלי
- 🇧🇷 **פורטוגזית** - שוק אמריקה הלטינית וברזיל
- 🇫🇷 **צרפתית** - הרחבה אירופית ושוק צרפתי
- 🇸🇦 **ערבית** - שוק ערבי מתפתח עם RTL מלא
- 🇮🇱 **עברית** - קהילת הטק ישראלית עם RTL
- 🇨🇳 **סינית** (סינית מפושטת) - ענק טכנולוגי אסייתי
